Suche

Unter "Spalte" kann die Spalte der Tabelle ausgewählt werden, die durchsucht werden soll.
Bei der Auswahl von "AuftragNummer" oder "Kundenname" wird das entsprechende Feld unter "Wert" zu einem Textfeld.
Bei der Auswahl von "Eingang" oder "Erteilt" wird das entsprechende Feld unter "Wert" zu zwei Datumsfeldern, in welche man Start und Ende eines Zeitraums eingeben kann.
Bei der Auswahl von "Erledigt" oder "Abgerechnet" wird das entsprechende Feld unter "Wert" zu einer Checkbox.
Verschiedene Filter können im Dropdown-Menü unter "Verknüpfung" durch "und" oder "oder" logisch miteinander verbunden werden.
Durch einen Klick auf den "Suchen"-Button wird die Suche gestartet.
